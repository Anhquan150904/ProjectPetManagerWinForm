Giải thích cho mọi người dễ hiểu:
  PetManagerData để xử lý dữ liệu, trong dự án này mình sẽ viết các hàm để lấy dữ liệu, thêm sửa xóa dữ liệu trong database, ở đây mình dùng database sql server
  PetManagerWinForm để mình làm winform thôi anh em, làm giao diện và xử lý các sự kiện, nếu phần nào cần làm việc với database thì mình sẽ gọi đến các hàm mình viết ở PetManagerData
  Đây mới chỉ là cấu trúc tạm thôi, để lên xem thầy có yêu cầu cấu trúc code không đã
  Với project này là tạm thôi, cho mọi người hiểu những gì cần làm, chứ chưa chốt đề tài
  Anh em sau này không hiểu gì cứ bảo t nhé, hoặc anh em làm WinForm thôi cũng được, còn backend dể mình t làm cũng được

Trong PetManagerData có file Database.script, anh em copy rồi chạy trên sql server để nó tạo mới database cho anh em, sau đó vào App.Config sửa lại connectString. 
Lưu ý anh em sửa trong db những gì thì copy lại hết ném vào Database.script rồi push code lên cho mọi người nhé, vì mình chạy db local nên phải chấp nhận vậy.

Anh em cứ đọc code phần đăng nhập đi, không hiểu chỗ nào nhắn trong nhóm hỏi t giải thích cho
Khi anh em code checkout đúng nhánh của mình nhé, t tạo sẵn mấy nhánh cho anh em rồi, code xong thì gửi push request, t xem chạy ổn thì t merge cho. Hôm nào gặp nhau hoặc vào discord t chỉ cách làm việc với github.
